HTML E SUAS TAGS

Tag < h >
< inicio

As tags < h1 > a < h6 > são utilizadas para titulo e subtitulo da página, respectivamente. Elas vão diminuido o tamanho da fonte conforme o número da tag aumenta.

Tag < p >
< inicio

A tag < p > é utilizada para definir um parágrafo.

Tag < mark >
< inicio

A tag < mark > é usada para marcar um trecho do texto como se utilizasse uma caneta marca texto.

Tag < small >
< inicio

A tag < small > é utilizada para fazer com que algum texto fique com um tamanho menor que as demais palavras do parágrafo.

Tag < i >
< inicio

A tag < i > é utilizada para fazer com que um texto especificado pela tag fique em itálico.

Tag < u >
< inicio

A tag < u > é utilizada para sublinhar um texto.

Tag < strong >
< inicio

A tag < strong > é utilizada para deixar um texto em negrito.

Tag < ol >
< inicio

A tag < ol > é utilizada para criar uma lista ordenada. Ela recebe este nome porque cada um de seus elementos recebem numeração. Por exemplo:
  1. Aline
  2. Bruna
  3. Gabriela

Tag < ul >
< inicio

A tag < ul > é utilizada para criar uma lista não-ordenada. Ela não recebe numeração em seus elementos, mas apenas pontos que os marcam. Por exemplo:
  • Café
  • Açucar
  • Arroz

Tag < a >
< inicio

A tag < a > é usada para criar um link na página. Ele pode redirecionar para outra página na mesma aba (que é o comportamento padrão), pode também direcionar na mesma aba através do atributo "target" com o valor "_blank". Também é possível direcionar para algum local especifico da mesma página do link, bastando apenas definir no atributo "href" da tag o "id" da tag especifica para a qual o link direcionará.

Tag < hr >
< inicio

A tag < hr > cria uma linha horizontal na página.

Tag < sub >
< inicio

A tag < sub > é utilizada para fazer com que algum caractere ou texto fique com um tamanho menor que os demais caracteres ou palavras do parágrafo. Por exemplo:
H 2 O = água.

Tag < sup >
< inicio

A tag < sup > é usada para colocar um caractere um pouco acima de outro caractere ou palavra. Por exemplo:
2 2 = 4.

Tag < blockquote >
< inicio

A tag < blockquote > é usada para citações.

Tag < font >
< inicio

A tag < font > é utilizada para definir uma determinada fonte, quer seja para texto ou cor.
Exemplos:
  1. Uma fonte disponível para uso é a Arial.
  2. A fonte a ser usada ficará destacada em vermelho.

Tag < del >
< inicio

A tag < del > é usada para especificar um trecho de texto que foi excluído.
Exemplo:
Esse texto foi excluído

Tag < abbr >
< inicio

A tag < abbr > é usada para explicar abreviações ou acrônimos ao passar o mouse por cima da abreviação. Ela deve vir acompanhada do atributo "title".
